# SnowRunner — Guides, Rankings, and Resources

This repository is a curated collection of community guides, rankings, spreadsheets, and video references for SnowRunner. The documents collected here are aimed at players who want to learn how trucks are classified, which trucks perform best, how tires work, which regions are most enjoyable or rewarding, and where to find detailed data spreadsheets and video walkthroughs.

Quick navigation — jump to a topic:
- <a>Truck Tier List.md</a> — Ranked truck lists with class-by-class breakdowns and timestamps.
- <a>Truck Classes.md</a> — Descriptions of truck classes (Highway, Heavy, Off-Road, Scout, etc.) and playstyle notes.
- <a>Tire Guide.md</a> — Deep dive on tire attributes, traction multipliers, recommended tires by truck type, choked/chained tires, and reference spreadsheets.
- <a>Snowrunner Region Rank.md</a> — Tiered ranking of all regions with short summaries, reasons for rank, and trucks unlocked.
- <a>Spreadsheets.md</a> — Curated links to community Google Sheets (tire comparisons, SR!NFO, USDS, and other data sources).
- <a>Youtube Videos.md</a> — Curated list of useful YouTube videos with timestamps: truck overviews, comprehensive tier lists, region rankings, and play order guides.

What each file contains (short summaries)

- Truck Tier List.md
  - Comprehensive ranking of trucks by performance tiers (S → D) with per-class sections (Scout, Off-Road, Heavy, Highway, etc.).
  - Includes timestamps and notes about each truck's strengths, weaknesses, and recommended uses.

- Truck Classes.md
  - Explains the different truck classes and how they affect role and performance in SnowRunner.
  - Practical notes on how to use each class effectively and common trade-offs.

- Tire Guide.md
  - In-depth explanation of tire attributes (traction multipliers, rolling resistance, flotation, and visuals).
  - Tire selection strategies organized by truck type, chaining recommendations, and a quick-reference table of recommended tires.
  - Links to the community tire spreadsheets used to derive the numbers and recommendations.

- Snowrunner Region Rank.md
  - Region-by-region tier list derived from video commentary and notes.
  - For each region: a short summary, reasons for its placement, and notable truck unlocks that make the region valuable (or not).

- Spreadsheets.md
  - Links to community-maintained Google Sheets for: tire comparisons, SnowRunner core data (SR!NFO), and larger aggregated datasets (USDS).
  - Good starting point if you want numerical values, raw stats, or to run your own comparisons.

- Youtube Videos.md
  - Curated set of video references with short descriptions and timestamps:
    - Truck tier guides and deep dives
    - Region rankings and play-order strategy videos
    - Other helpful visual walkthroughs that complement the written guides

How to use this repo (recommended reading order)
1. New player: Truck Classes → Truck Tier List → Tire Guide → Region Rank
2. Optimize trucks/gear: Tire Guide → Spreadsheets → Truck Tier List
3. Plan play order / region run: Snowrunner Region Rank → Youtube Videos
